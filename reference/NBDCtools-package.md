# NBDCtools: National Institutes of Health Brain Development Cohorts Data Hub Tools

A suite of functions to work with data from the National Institutes of
Health Brain Development Cohorts Data Hub. The package provides tools to
create, clean, process, and filter datasets and associated metadata.
These utilities are intended to simplify reproducible data-preparation
for future research.
